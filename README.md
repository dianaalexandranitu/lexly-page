# Lexly — Legal documents for Dutch startups

A free tool that generates Terms & Conditions, Privacy Policy, AI Act compliance notes, and Cookie Policy for companies incorporated in the Netherlands.

## How it works

1. Answer ~25 questions about your business
2. Enter a free Anthropic API key
3. Receive complete legal documents with every clause explained in plain English

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Deploy to GitHub Pages (free, ~5 minutes)

1. Create a GitHub repository
2. Push this code:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
   git push -u origin main
   ```
3. Build the project:
   ```bash
   npm run build
   ```
4. Deploy the `dist/` folder to GitHub Pages:
   - Go to your repo → Settings → Pages
   - Source: Deploy from a branch
   - Branch: `gh-pages` (or use the `dist` folder on `main`)
   
   Or use the `gh-pages` package for one-command deploys:
   ```bash
   npm install --save-dev gh-pages
   ```
   Add to `package.json` scripts: `"deploy": "npm run build && gh-pages -d dist"`
   Then: `npm run deploy`

Your site will be live at: `https://YOUR_USERNAME.github.io/YOUR_REPO/`

## API key

Users need a free Anthropic API key from https://console.anthropic.com
The key is never stored — it's used only client-side for the generation request.

## Tech stack

- React 18 + Vite
- CSS Modules
- Anthropic Claude API (claude-opus-4-5)
- No backend, no database, no accounts
